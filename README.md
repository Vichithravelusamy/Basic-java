# Student Management System

A console-based Student Management System built using Java.

## 📌 About the Project

This project is used to manage student records. 
It allows users to add, view, search, and delete student details.
The data is stored permanently using file handling.

## 🚀 Features

- Add Student
- View All Students
- Search Student by Roll Number
- Delete Student
- Permanent data storage
- Simple menu-driven console interface

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- File Handling
  
## 📂 Project Structure

Student-Management-System

│── Main.java

│── Student.java

│── StudentManager.java

│── students.txt

│── README.md


## ▶️ How to Run

1. Compile the Java files:
javac Main.java

2. Run the program:
java Main
